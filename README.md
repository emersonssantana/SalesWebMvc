# SalesWebMvc

A web application built with ASP.NET Core MVC to manage departments, sellers, and sales records.

## Technologies
* C# / .NET
* ASP.NET Core MVC
* Entity Framework
* MySQL
* Bootstrap

## Features
* Department management
* Seller registration
* Sales tracking and recording

## How to run
1. Clone the repository.
2. Configure your database connection in `appsettings.json`.
3. Run the application using Visual Studio or the .NET CLI.
